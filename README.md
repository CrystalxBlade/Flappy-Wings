## 💭 About the Game

Flappy Wings is a 2D endless arcade game developed in Unity, inspired by the classic Flappy Bird experience.

Players control a bird by tapping to flap upward while navigating through an endless series of pipes. The objective is simple: survive as long as possible and achieve the highest score without colliding with obstacles.

---

## ✨ Features

- Responsive tap-to-flap controls
- Rigidbody2D based bird movement
- Procedural pipe spawning system
- Score tracking and UI updates
- Sound effects for flap, scoring, and collisions
- Game Over screen with restart functionality
- Pause and resume system
- Android support

## 🚀 How to Play
```
1. Download the apk from itch.io  
2. Install the apk file  
3. Run `Flappy Wings`

⚠️ Android might act a little dramatic and say “this app looks suspicious” 😤  
Don’t worry — it’s just because the game is new.

👉 ** → Run anyway** and enjoy!
```
